# Meridian Airlines — Corporate Fare Benefits Guide
**Policy Reference:** MA-CO-002
**Effective Date:** 1 November 2025
**Applies To:** Organisations with a valid Meridian Corporate Account (MA-CORP-XXXXXXXX)

---

## 1. Corporate Fare Classes

Corporate fares are identified by the **MA-CN** fare code prefix. The following corporate fare classes are available:

| Corporate Fare Class | Based On | Key Benefit |
|---|---|---|
| MA-CN-S (Saver) | Economy Saver | Discounted rate; free changes |
| MA-CN-Y (Flex) | Economy Flex | Discounted rate; free changes; full refund |
| MA-CN-C (Business) | Business | Discounted rate; all Business benefits |
| MA-CN-F (First) | First | Negotiated rates for qualifying accounts |

---

## 2. Ticket Flexibility Under Corporate Fares

Corporate fares override the standard fare class rules for changes and cancellations:

| Rule | Standard Fare | MA-CN Corporate Fare |
|---|---|---|
| Date/time change fee | USD 0–200 (fare-dependent) | USD 0 (free, unlimited) |
| Route change fee | USD 75–150 | USD 0 |
| Cancellation fee | USD 0–200 | USD 0 (fully refundable) |
| Refund type | Credit or cash (fare-dependent) | Always cash to original payment |
| No-show penalty | Up to full fare | USD 0 (full refund) |

---

## 3. Meridian Miles Earning on Corporate Fares

Employees travelling on corporate fares earn Meridian Miles to their personal accounts (MA-XXXXXXXX) at enhanced rates:

| Corporate Fare Class | Miles Earning |
|---|---|
| MA-CN-S | 75% of distance (vs 25% for standard Economy Saver) |
| MA-CN-Y | 125% of distance (vs 100% for standard Economy Flex) |
| MA-CN-C | 175% of distance (vs 150% for standard Business) |
| MA-CN-F | 225% of distance (vs 200% for standard First) |

Enhanced earning applies to the individual traveller's Meridian Miles account. Corporate accounts also earn **Company Miles** (see Section 4).

---

## 4. Company Miles Programme

In addition to individual Meridian Miles, Meridian corporate accounts earn **Company Miles** that belong to the organisation:

| Earning Rate | 1 Company Mile per USD 5 of qualifying spend |
|---|---|
| Redemption | Meridian travel credits (MA-TC-XXXXXX) redeemable by any employee |
| Validity | 36 months from earning date |
| Reporting | Visible in the corporate dashboard at meridianair.com/corporate/dashboard |

Company Miles cannot be transferred to individual Meridian Miles accounts.

---

## 5. Priority Services on Corporate Fares

Travellers on MA-CN fares receive the following priority services at no extra charge:

| Service | MA-CN-S | MA-CN-Y | MA-CN-C | MA-CN-F |
|---|---|---|---|---|
| Priority check-in | Yes | Yes | Yes | Yes |
| Priority boarding | No | Yes | Yes | Yes |
| Priority baggage | No | Yes | Yes | Yes |
| Lounge access | No | No | Yes (Horizon) | Yes (First) |
| Fast-track security | No | Yes | Yes | Yes |

---

## 6. Travel Policy Compliance Features

Meridian's corporate account portal provides tools to enforce company travel policies:

| Feature | Description |
|---|---|
| Booking class restrictions | Restrict employees to specific cabin classes |
| Fare cap enforcement | Set maximum fare per route |
| Advance purchase requirements | Flag bookings made within X days of departure |
| Approval workflows | Route certain bookings for manager approval |
| TMC integration | Compatible with Concur, Amadeus, Sabre, and Galileo |

These features are configured in the corporate dashboard. Contact the Meridian Account Manager (MA-CO-001) to configure restrictions.
