# Meridian Airlines — Travel Management Guide
**Policy Reference:** MA-CO-003
**Effective Date:** 1 November 2025
**Applies To:** Corporate travel managers and TMCs managing Meridian Airlines accounts

---

## 1. Travel Management Company (TMC) Integration

Meridian Airlines supports booking and management through all major Travel Management Companies (TMCs) and Global Distribution Systems (GDS).

### 1.1 Supported GDS Platforms
| Platform | Meridian Code | Full Content Available |
|---|---|---|
| Amadeus | MA | Yes |
| Sabre | MA | Yes |
| Galileo / Travelport | MA | Yes |
| Worldspan | MA | Yes |

### 1.2 NDC (New Distribution Capability) Content
Meridian Airlines supports IATA NDC Level 4 content distribution. NDC-enabled bookings provide access to:
- Ancillary services (seats, bags, lounge passes) bundled in the booking flow
- Corporate negotiated fares with dynamic pricing
- Real-time seat maps and availability
- Rich media content (aircraft photos, meal descriptions)

NDC-certified agency list: meridianair.com/ndc-partners.

## 2. Online Booking Tool (OBT) Integration

Meridian Airlines is compatible with major OBT platforms used by corporate travel managers:

| OBT Platform | Integration Status |
|---|---|
| SAP Concur Travel | Full integration (real-time fares + ancillaries) |
| Cytric by Amadeus | Full integration |
| KDS Neo | Full integration |
| Deem | Full integration |
| TravelPerk | API integration (fares only) |

To configure your OBT to display Meridian corporate fares, enter the Corporate Account Number (MA-CORP-XXXXXXXX) in the Meridian supplier settings within the OBT.

## 3. Corporate Dashboard Features

The Meridian Corporate Dashboard (meridianair.com/corporate/dashboard) provides:

### 3.1 Reporting
| Report Type | Frequency | Format |
|---|---|---|
| Travel spend by employee | Monthly / on-demand | PDF, CSV |
| Travel spend by route | Monthly / on-demand | PDF, CSV |
| CO₂ emissions per trip | Monthly / on-demand | PDF, CSV |
| On-time performance for corporate routes | Monthly | PDF |
| Unused ticket report | Monthly | PDF, CSV |

### 3.2 Booking Management
- View and modify all open bookings across the account
- Cancel on behalf of travellers
- Rebook disrupted travellers
- Bulk itinerary changes for group travel

### 3.3 Traveller Profiles
- Store passport details, meal preferences, seat preferences for frequent travellers
- Auto-populate at booking to speed up the reservation process
- Compliance flags for travellers exceeding fare caps

## 4. Duty of Care and Traveller Tracking

Meridian Airlines provides a **Traveller Tracking API** for corporate accounts:

| Feature | Details |
|---|---|
| Real-time flight status | JSON API, push or pull |
| Passenger manifest | Available for corporate bookings on request |
| Emergency contact integration | Link to your HR system or duty of care platform |
| Disruption alerts | Automated email/SMS to travel manager when a Meridian flight is disrupted |

Duty of care integration: api.meridianair.com/corporate/tracking. API key issued upon request to corporate@meridianair.com.

## 6. Programme Performance and Discount Unlock

Corporate accounts are evaluated annually. Discount tiers unlock based on **achieved annual spend**:

| Achieved Spend | Next Tier Discount | Review Date |
|---|---|---|
| Reaches USD 150K | Business discount increases by 3% | 1 January each year |
| Reaches USD 500K | Bespoke negotiation with Account Manager | Triggered immediately |

If the annual spend target is not met, the discount tier is renegotiated at contract renewal. Meridian Airlines may adjust the discount tier downward if spend falls by more than 25% from the prior year.
